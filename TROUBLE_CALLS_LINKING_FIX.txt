FIX #1: Made Trouble Call IDs Clickable in list.php
────────────────────────────────────────────────────

Modified: list.php (lines 163-168 and rows processing section)

Changes:
  - Added detection for 'hj_id' column in query results
  - Wrapped hj_id values with HTML anchor tags linking to hot_job.php
  - Each trouble call ID now displays as a blue, underlined link
  - When clicked, opens: hot_job.php?hj_id=<number>

Code Logic:
  ```php
  // Detect if this is a trouble calls list
  $has_hj_id = !empty($all_rows) && isset($all_rows[0]['hj_id']);
  $hj_id_col_index = $has_hj_id ? array_search('Trouble Call', $table->col_names) : -1;
  
  // In row processing loop:
  if ($has_hj_id && $hj_id_col_index >= 0) {
      $hj_id = $all_rows[$i]['hj_id'] ?? '';
      if ($hj_id) {
          $row_data[$hj_id_col_index] = "<a href='hot_job.php?hj_id=$hj_id'>$hj_id</a>";
      }
  }
  ```

Result: ✓ Clicking on Trouble Call ID now opens detail page
