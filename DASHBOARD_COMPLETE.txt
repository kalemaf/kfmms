## 🔍 Troubleshooting

### Dashboard shows "No data available"
- Check if you have work orders in the database
- Verify work order status values match: 'New', 'Assigned', 'In Progress', 'Completed', 'Cancelled'

### Charts not rendering
- Check browser console (F12) for JavaScript errors
- Verify Chart.js CDN is accessible: https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js

### Styling issues
- Clear browser cache (Ctrl+Shift+Del)
- Verify Bootstrap CDN is loaded: https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css

## 📞 Support Notes
