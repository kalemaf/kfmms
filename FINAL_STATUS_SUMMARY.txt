2. NEW WORK ORDER CREATED
   INSERT INTO work_orders (..., tenant_id) 
   VALUES (..., 1)  ← automatically from session

3. VIEW WORK ORDERS
   SELECT * FROM work_orders 
   WHERE tenant_id = 1  ← automatically filtered

4. RESULT
   User only sees Company A's work orders

5. SWITCH TO COMPANY B
   Session: $_SESSION['tenant_id'] = 31
   All queries automatically filtered to tenant 31
   Company A's data completely hidden
