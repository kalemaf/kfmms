## Consumables Location Dropdown Flow

```
┌─────────────────────────────────────────────────┐
│       CONSUMABLES.PHP - Add/Edit Form            │
├─────────────────────────────────────────────────┤
│                                                 │
│  Location Field:                                │
│  [Select warehouse location ▼]                  │
│                                                 │
│  Before: Text input → "Warehouse 1, Zone A"    │
│  After:  Dropdown → Shows formatted options    │
│                                                 │
└────┬────────────────────────────────────────────┘
     │
     ↓ Load warehouse locations
┌─────────────────────────────────────────────────┐
│  inventory_manager.php                          │
│  get_all_warehouse_locations($connection)       │
├─────────────────────────────────────────────────┤
│  SELECT wl.*, w.warehouse_name                  │
│  FROM warehouse_locations wl                    │
│  LEFT JOIN warehouses w                         │
│  WHERE wl.is_active = 1                         │
│  Apply tenant filter ➜ tenant_id = {current}   │
└────┬────────────────────────────────────────────┘
     │
     ↓ Return array of locations
┌─────────────────────────────────────────────────┐
│  Dropdown Rendering                             │
├─────────────────────────────────────────────────┤
```

## Query Filtering Logic

```
QUERY TRANSFORMATION EXAMPLE:

Original Query (No Filtering):
┌────────────────────────────────────────┐
│ SELECT * FROM work_orders              │
│ ORDER BY submit_date DESC              │
└────────────────────────────────────────┘
         │
         │ apply_tenant_filter()
         ↓

Tenant 1 Query (Session tenant_id = 1):
┌────────────────────────────────────────────────────┐
│ SELECT * FROM work_orders                          │
│ WHERE tenant_id = 1                                │
│ ORDER BY submit_date DESC                          │
│                                                    │
│ Result: WO #1, #2, #3, #4, #5 (5 records)         │
└────────────────────────────────────────────────────┘

Tenant 31 Query (Session tenant_id = 31):
┌────────────────────────────────────────────────────┐
│ SELECT * FROM work_orders                          │
│ WHERE tenant_id = 31                               │
│ ORDER BY submit_date DESC                          │
│                                                    │
│ Result: WO #6 (1 record)                           │
└────────────────────────────────────────────────────┘

Tenant 32 Query (Session tenant_id = 32):
┌────────────────────────────────────────────────────┐
│ SELECT * FROM work_orders                          │
│ WHERE tenant_id = 32                               │
│ ORDER BY submit_date DESC                          │
│                                                    │
│ Result: WO #7 (1 record)                           │
└────────────────────────────────────────────────────┘
```
