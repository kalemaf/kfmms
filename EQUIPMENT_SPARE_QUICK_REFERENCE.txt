# Equipment Spare Association Feature - Quick Reference

## What Was Added ✅

A new **"Equipment Compatibility & Spares"** section in the Parts Master form that allows you to:
- Attach parts to specific equipment items
- Automatically store parts as spares in equipment inventory
- Easily assess and monitor spares per equipment
- Track which equipment uses each spare part

---

## Key Features

### ✓ Multi-Select Equipment Checkboxes
- Select one part for multiple equipment
- Show manufacturer, model, and serial number
- Scrollable list for large equipment inventories

### ✓ Visual Management
- Info banner explaining the feature
- Checked equipment highlighted  
- Associated equipment cards (green section)
- Search/filter capability

### ✓ Smart Syncing
- New equipment automatically linked
- Deselected equipment relationships removed
- Part data cached in equipment_spares
- Integrates seamlessly with inventory system

---

## How to Use It

### Adding Equipment Spares to a Part

1. **Go to:** Inventory → Parts Master → New Part
2. **Fill in part details** (Part Code, Part Name, etc.)
3. **Scroll to:** "Equipment Compatibility & Spares" section
4. **Check boxes** for each equipment this spare fits
   - If no equipment available, add equipment first
5. **Click** "Save Part"
6. ✅ Part automatically becomes a spare for selected equipment

### Managing Existing Associations

1. **Go to:** Inventory → Parts Master → Edit Part
2. **Scroll to:** "Equipment Compatibility & Spares"
3. **View** currently associated equipment (green box)
4. **Modify** selections:
   - Check to add equipment
   - Uncheck to remove equipment
5. **Click** "Save Part"
6. ✅ Associations automatically updated

## What Happens When You Save

```
Select Equipment & Save Part
        ↓
System creates/updates parts_master record
        ↓
System syncs equipment_spares table:
  • Adds newly selected equipment
  • Removes deselected equipment
  • Keeps existing associations
        ↓
Part automatically appears in:
  • Equipment spares lists
  • Inventory dashboard
  • Work order spare lookups
  • Low-stock alerts
```

## Example Workflow

**Scenario:** You have a Pump (Pump-101) that requires 2 roller bearings and 1 coupling

1. **Create Part:** "6208ZZ Roller Bearing"
2. **Scroll to Equipment section**
3. **Check:** Pump-101
4. **Save** → Spare created for Pump-101

5. **Create Part:** "Motor Coupling ABC"
6. **Scroll to Equipment section**
7. **Check:** Pump-101
8. **Save** → Spare created for Pump-101

**Result:** Pump-101 now shows 2 spares available in the system!

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Equipment section is empty | Add equipment first in Equipment Management |
| Can't find a part after saving | Verify part is marked as "Active" |
| Associations not saved | Check part Code is unique and Name filled in |
| Spares not showing in inventory | Verify equipment_spares table exists in database |
